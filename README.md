# Clone Slack-чат

[![Render](https://img.shields.io/badge/Render-Deployed-brightgreen)](https://hexlet-chat-a1xt.onrender.com/)

## О проекте

**Clone Slack-чат** — приложение для обмена сообщениями в реальном времени, упрощенный аналог популярного мессенджера Slack.

### Предварительные требования

Убедись, что у тебя установлены:

- Node.js
- npm или yarn

### Установка и запуск

1. **Установка зависимостей:**

```

npm install

```

2. **Запуск:**

```

make develop

```
